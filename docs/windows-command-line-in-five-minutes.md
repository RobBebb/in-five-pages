## Commands

A list of commands/programs that work for me. Some may be due to me having git bash installed.
|Keys|Description|
-|-
`help`|Get a list of commands available
`cat xxx.txt`|Display file xxx.txt
`del xxx.txt`|Delete file xxx.txt
`ls`|List folders and files in a directory
`cd bin`|Change to the bin directory
`notepad xxx.txt`|Edit file xxx.txt in notepad
